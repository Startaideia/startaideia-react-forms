Installation

Styled Components#

To use @startaideia/react-forms you must install styled-components in your react application.

#NPM
npm install styled-components
#Yarn
yarn add styled-components

With styled-components installed, run the following commands to add @startaideia/react-forms to your application

# NPM
npm install @startaideia/react-forms
#Yarn
yarn add @startaideia/react-forms

And it is done! It is all you need to start using @startaideia/react-forms. See examples and how to in the next sections
